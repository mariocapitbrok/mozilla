Character Classes.

\d - Matches any digit character (0-9).
\D - Matches any non-digit character.
\w - Matches any word character (i.e., alphanumeric characters and underscores).
\W - Matches any non-word character.
\s - Matches any whitespace character (i.e., spaces, tabs, and newlines).
\S - Matches any non-whitespace character.
[abc] - Matches any of the characters within the brackets (i.e., a, b, or c).
[^abc] - Matches any character that is not within the brackets (i.e., any character except a, b, or c).
[a-z] - Matches any character within the range of a to z (i.e., lowercase letters).
[A-Z] - Matches any character within the range of A to Z (i.e., uppercase letters).
[0-9] - Matches any digit character (same as \d).
[^\d] - Matches any non-digit character (same as \D).
[\w-] - Matches any word character or hyphen character.
